# Example

Run locally:

1. `pnpm install`
2. `pnpm dev`
3. open `http://localhost:4321/`

Build:

- `pnpm build`

Pages:

- `/` integration overview
- `/hot-feature-1/` `hotFeature1` route + shorthand
- `/hot/` `hotFeature2` route + shorthand
- `/hot-dev/sub/` wildcard nested route + combined `dev` + `hotFeature2` gating
