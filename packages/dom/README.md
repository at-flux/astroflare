# @at-flux/dom

[![npm version](https://img.shields.io/npm/v/@at-flux/dom.svg)](https://www.npmjs.com/package/@at-flux/dom)
[![CI](https://github.com/at-flux/astroflare/actions/workflows/ci.yml/badge.svg)](https://github.com/at-flux/astroflare/actions/workflows/ci.yml)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Type-safe DOM helpers for Astro / browser code. **Separate** from `@at-flux/astroflare` — add it as its own dependency and import from `@at-flux/dom`.

## Install

```bash
pnpm add @at-flux/dom
```

## Usage

```ts
import { getElementById, getElementByQueryOrThrow } from "@at-flux/dom";
```
